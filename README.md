# Mini Banking System

A simple interactive banking system built with HTML, CSS, and JavaScript.  
Users can deposit, withdraw, and track their transaction history dynamically.

---

## Features

- Deposit & Withdraw functionality
- Balance validation (prevents overdraft)
- Dynamic transaction history tracking
- Interactive UI built with vanilla JavaScript
- Clean, professional design with CSS

---

## Technologies Used

- HTML
- CSS
- JavaScript
